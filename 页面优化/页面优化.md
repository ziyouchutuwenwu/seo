# 页面优化

## 优化手段

### 通用

```sh
title, h1, h2 都可以增加关键词的权重，一个关键字，用一个页面单独做
title 最重要，首页 title 里面，放最重要的几个关键字，其它放到页面，一个页面一个 title
description, 搜索的时候会显示出来
一个页面只应放一个 h1 标签，h1 标签与页面开头的权重几乎相同
h2, h3 标签都是用来进一步解释 h1 标签的内容
每个图片的 alt 是图片描述，给搜索引擎看的，非常重要；title 辅助
```

### 超链接

```html
rel="noopener noreferer nofollow"
```

```sh
noopener  不让操作 window.open
noreferer 这里点出去，看不到 refer
nofollow  网站的权重不让传递出去
```

### canonical

header 里面，防止权重分散

从其它地方跳过来的网址，无论是 `https://aaa.com/?aa=xxx` 还是 `https://aaa.com/?zz=bbb`，网址应该都是 `https://aaa.com`

```sh
<link rel="canonical" href="https://aaa.com">
```
