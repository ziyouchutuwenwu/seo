# 衡量关键词

## 说明

判断关键词是否值得做

## 步骤

### 衡量公式

kdroi: 优化难度回报率

```sh
kdroi= volumn * cpc / kd
```

```sh
kdroi   优化难度回报率
volumn  关键词的搜索量
cpc     关键词的点击单价
kd      关键词优化难度
```

### 操作步骤

[semrush](https://www.semrush.com/analytics/keywordmagic/start) 里面获取关键字的 kdroi
