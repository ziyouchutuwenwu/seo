# 收入估算

## 说明

| 指标名称 | 定义描述                                               |
| -------- | ------------------------------------------------------ |
| UV       | 独立访客量                                             |
| PV       | 页面访问量，一个 uv 可能产生多次 pv                    |
| 广告曝光 | 网页里的广告被展示的次数                               |
| ECPM     | 广告展示 1000 次的收入，国内不到 1 美元，国外 3-6 美元 |

ECPM 离钱越近，单价越高

## 计算公式

### 单次估算

假定

```sh
日 UV 1000
ECPM  1 美元
UV    1 个PV
PV    1 次广告展示
```

则日收入为 1 美元，月收入 30 美元

### 实际计算

假如，欧美查询类工具网站

```sh
# 10 倍的千次数
日 UV 10000
UV    3 个PV
PV    3 次广告
ECPM  4 美元
```

收入估算

```sh
30 美金 *3*3*4*10
```
